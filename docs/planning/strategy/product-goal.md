# Product Goal — AI Platform for Smart Home Assistants

## 18-month vision (Target State)

AI Platform — это контрактно-строгая, управляемая платформа принятия решений,
которая позволяет продуктам создавать “умных помощников” на базе LLM
без потери предсказуемости, безопасности и контроля качества.

Платформа:
- принимает естественные команды пользователей,
- нормализует и интерпретирует их,
- возвращает структурированное решение (DecisionDTO),
- **не выполняет доменные действия напрямую**,
- масштабируется на несколько продуктов и доменов.

## Primary consumer (initial)

- HomeTusk — бытовые задачи и покупки для домохозяйств.

## Core problems we solve

1. Непредсказуемость LLM в проде
2. Смешение ответственности “ИИ vs продукт”
3. Сложность эволюции логики без поломки UX и контрактов
4. Отсутствие измеримости “умности”

## Product pillars

1. **Contract-first decisioning**
   - стабильные входы/выходы
   - строгая совместимость версий

2. **Deterministic by default**
   - LLM усиливает, но не заменяет правила

3. **Explainability & observability**
   - трассируемые решения
   - измеримая польза LLM

4. **Safe extensibility**
   - новые продукты и интенты без переписывания платформы

## Anti-scope (не является целью)

- Универсальный чат-бот
- Исполнение бизнес-действий внутри платформы
- Автономные агенты без продуктового контроля
- “Долгая память” и персональные профили в MVP

## Success metrics

### North Star
- Доля пользовательских команд, завершившихся `start_job` без дополнительных clarify

### Supporting metrics
- Среднее число clarify-итераций на команду
- Intent match rate (LLM vs deterministic baseline)
- p95 latency решения
- Доля LLM-ошибок с безопасным fallback

## Governance

- Любая инициатива должна маппиться на один из Product Pillars
- Изменения, влияющие на контракты или decision flow, требуют ADR
