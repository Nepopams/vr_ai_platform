# ADR-003 (Draft): Экспериментальный LLM shopping-list extractor за флагами

- Status: Draft
- Date: 2026-01-12
- Owners: Platform Team

## Контекст

MVP v1 и ADR-002 запрещают внешние LLM-вызовы и реальные execution на стороне платформы.
При этом нужно провести ограниченную оценку качества извлечения покупок с использованием
локального agent runner, не меняя DecisionDTO и не влияя на боевой runtime.

## Решение (Draft)

Добавить локальный agent runner как экспериментальный сервис, полностью выключенный
по умолчанию и включаемый только через флаги окружения. Результаты используются
исключительно в shadow-mode для логирования и сравнения, без влияния на DecisionDTO.

Shadow Router (наблюдаемость) работает в best-effort режиме и запускается
параллельно RouterV2. Он является internal-only механизмом, не влияет на
DecisionDTO и не меняет публичные контракты.

## Ограничения

- Никаких изменений contracts/graphs/DecisionDTO.
- По умолчанию LLM выключен.
- Результаты LLM не влияют на решения.
- Нет новых intents/actions/полей без отдельного ADR и semver-bump.

## Статус

Draft до принятия решения о расширении MVP.
